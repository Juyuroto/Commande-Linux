## 1. Création et manipulation de fichiers

### Créez une structure de projet

**À faire :**

- Créez un dossier "mon_projet" dans votre home
- Dedans, créez 3 sous-dossiers : src, docs, tests 
- Dans src, créez 2 fichiers : main.py et utils.py 
- Copiez main.py vers main.backup.py 
- Listez tout récursivement

### Solution

```bash
mkdir mon_projet
cd mon_projet
mkdir src docs tests
cd src
touch main.py utils.py
cp main.py main.backup.py
ls -R
```
